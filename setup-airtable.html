Crear base de datos Airtable

Este script crea la tabla de leads con todos los campos (formulario + análisis IA) en tu base de Airtable. Ejecútalo una sola vez.

Consíguelo en airtable.com/create/tokens → Create token → Scope: schema.bases:write + data.records:write

Crea una base vacía en Airtable llamada "tenerifIA CRM". El ID está en la URL: airtable.com/appXXXXXX/...
